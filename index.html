Notebook App

This Notebook App, built with the MERN (MongoDB, Express, React, Node.js) stack, allows users to perform full CRUD (Create, Read, Update, Delete) operations on their notes, with all data being securely stored in a database. Users can create and edit notes, update content as needed, and delete notes when they are no longer relevant. While the app lacks a search functionality, it provides a straightforward way to manage notes efficiently.Key features include database integration for persistent data storage, real-time note editing, and a user-friendly interface. The app ensures smooth performance and responsiveness across devices, making it easy for users to manage their notes from anywhere. This project demonstrates skills in connecting a React frontend with a Node.js and Express backend, along with MongoDB for data storage.
